# ChangNameVideo 🎬

โปรแกรมเปลี่ยนชื่อวิดีโอแบบสุ่ม พร้อมแฮชแท็ก

## วิธีใช้งาน (Windows)

1. ดับเบิลคลิก `run.bat`
2. หรือใช้คำสั่ง:
   ```
   python ChangNameVideo.py
   ```

## วิธีใช้งาน (Mac/Linux)

```bash
chmod +x run.sh
./run.sh
```

## ขั้นตอนการใช้งาน

1. **เลือกไฟล์วิดีโอ** — กด "เลือกไฟล์วิดีโอ" หรือ "เลือกทั้งโฟลเดอร์"
2. **ใส่ชื่อวิดีโอ** — ใส่ชื่อที่ต้องการ 1 ชื่อต่อ 1 บรรทัด (ช่องซ้าย)
3. **ใส่แฮชแท็ก** — ใส่แฮชแท็ก 1 อันต่อ 1 บรรทัด (ช่องขวา)
4. **ตั้งค่าตัวเลือก** — จำนวนแท็ก, คำค้น, ตำแหน่งแท็ก, สุ่มหรือไม่
5. **กดพรีวิว** — ดูตัวอย่างชื่อใหม่ก่อนเปลี่ยนจริง
6. **กดเริ่มเปลี่ยนชื่อ** — ยืนยันแล้วเปลี่ยนชื่อจริง

## ต้องการ

- Python 3.7+
- tkinter (ติดตั้งมาพร้อม Python แล้ว)
- ไม่ต้องติดตั้ง library เพิ่มเติม
