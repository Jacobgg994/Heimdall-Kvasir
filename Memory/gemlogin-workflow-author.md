Every GemLogin workflow must use:
- **author**: `"GemLogin Team"`
- **description**: `"Built by GemLogin Team: JACOB (CEO) | GEMMY (Automation Lead) | ARKA (Workflow Creator) | LUMI (Workflow Analyst) | HEIMDALL (QA Engineer) | JIMMY (COO)"`

**Why:** User wants consistent team attribution on all workflows.
**How to apply:** When building or updating any GemLogin workflow, set `author` and `description` fields to these values.
