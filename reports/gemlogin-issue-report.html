Proxy ฝัง Windows ไม่ได้ อินฟลู ฝัง WiFi ใช้ได้ ✓ / ฝัง Windows ไม่ได้ ✗ (แม้ติดตั้งแอพแล้ว) — ต้องใช้แอพแปลง Proxy
