## 🎯 กลุ่มลูกค้าเป้าหมาย

### กลุ่ม 1: นักการตลาดออนไลน์ (ปิดง่าย — Quick Win)
- **ใคร:** คนทำ Affiliate, TikTok Shop, Facebook Ads, คนที่ใช้หลายบัญชี
- **ปัญหา:** จัดการหลายบัญชี — โดนแบนบ่อย — เสียเวลา login/logout
- **แนะนำ:** GEM200 (฿450/เดือน) หรือ Unlimited 1M (฿700/เดือน)
- **ช่องทาง:** Facebook Groups, TikTok, LINE OA
- **ระยะเวลาปิด:** 1-7 วัน

### กลุ่ม 2: เอเจนซี่การตลาด (มูลค่าสูง)
- **ใคร:** Digital Agency, Social Media Agency, Media Buyer
- **ปัญหา:** ทีม 5-10 คน จัดการบัญชีลูกค้า 100+ บัญชี — ไม่มีระบบ — ข้อมูลกระจัดกระจาย
- **แนะนำ:** Cloud Professional ($99/เดือน) หรือ Enterprise
- **ช่องทาง:** LinkedIn, LINE OA, ลูกค้าแนะนำ, งาน Event
- **ระยะเวลาปิด:** 2-4 สัปดาห์

### กลุ่ม 3: เจ้าของร้านค้าออนไลน์ (ปริมาณมาก)
- **ใคร:** ร้านค้า Shopee/Lazada, Dropshipper, TikTok Shop Seller
- **ปัญหา:** ต้องติดตามราคาคู่แข่ง — ต้องสร้างรีวิว — จัดการหลายร้าน
- **แนะนำ:** Unlimited 1M (฿700/เดือน) + Script เสริม
- **ช่องทาง:** TikTok, Facebook Groups E-commerce
- **ระยะเวลาปิด:** 1-2 สัปดาห์

### กลุ่ม 4: นักพัฒนา (สร้างระบบนิเวศ)
- **ใคร:** โปรแกรมเมอร์, Automation Builder, Freelancer
- **ปัญหา:** ต้องการแพลตฟอร์มขาย Script — ต้องการรายได้เสริม
- **แนะนำ:** GemStore + ส่วนแบ่งรายได้ 70%
- **ช่องทาง:** GitHub, Dev.to, Facebook Groups โปรแกรมเมอร์
- **ระยะเวลาปิด:** 1-4 สัปดาห์

## 🔄 กระบวนการขาย

```
ขั้นที่ 1: หาลูกค้า
├── ขาเข้า: LINE OA, คอมเมนต์ Facebook, TikTok
├── ขาออก: DM, ข้อความหาลูกค้า, อีเมล
└── แนะนำ: Affiliate, ลูกค้าเก่า

ขั้นที่ 2: คัดกรอง (BANT)
├── งบประมาณ: มีงบไหม?
├── อำนาจตัดสินใจ: คนคุยตัดสินใจซื้อได้ไหม?
├── ความต้องการ: ใช้กี่บัญชี? แพลตฟอร์มอะไร?
└── ระยะเวลา: ต้องการใช้เมื่อไหร่?

ขั้นที่ 3: สาธิต / ทดลองใช้
├── แนะนำแพ็กเกจฟรี (5 โปรไฟล์ตลอดชีพ)
├── สอนใช้งาน 15-30 นาที
└── ส่งวิดีโอสอน + คู่มือ

ขั้นที่ 4: ยื่นข้อเสนอ
├── แนะนำแพ็กเกจที่เหมาะ
├── คำนวณความคุ้มค่า (ROI)
├── เทียบคู่แข่ง (ถ้าจำเป็น)
└── เสนอทางเลือก A/B/C

ขั้นที่ 5: ปิดการขาย
├── จัดการข้อกังวล
├── ปิดดีล
├── แจ้งวิธีชำระเงิน (Coin/โอนผ่านธนาคาร)
└── สอนเริ่มใช้งาน

ขั้นที่ 6: ต่อยอด
├── ติดตามวันที่ 1, 3, 7
├── ขายเพิ่ม (Local → Cloud, GemLogin → GPF)
├── ขอคำนิยม / เรื่องราวความสำเร็จ
└── ชวนเข้าโปรแกรม Affiliate
```
