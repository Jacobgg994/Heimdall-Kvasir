# 🎨 Content Templates — SALMON

## TikTok (15-60 วิ)

### Hook Templates
```
🔥 "รู้หรือยัง..." — curiosity gap
😱 "สิ่งที่ [แบรนด์] ไม่เคยบอกคุณ" — secret reveal
📊 "3 วิธีที่..." — listicle
⚡ "ดูนี่ก่อน..." — urgency
🤯 "เปลี่ยน [ปัญหา] เป็น [ผลลัพธ์] ใน [เวลา]" — transformation
```

### Structure
```
Hook (3 วิ) → Problem (5 วิ) → Solution (10-20 วิ) → Result (5 วิ) → CTA (3 วิ)
```

### Caption
```
[hook ซ้ำ] + [context 1-2 บรรทัด] + [hashtag 3-5 ตัว] + [CTA]
```

## Facebook Post

### ประเภท
| ประเภท | ความยาว | ใช้เมื่อ |
|--------|--------|---------|
| Short | 1-3 ประโยค | Engagement, quick tip |
| Medium | 3-5 บรรทัด | Story, case study |
| Long | 5-10 บรรทัด | Tutorial, review |

### Template
```
[ประโยคเปิด — hook ดึงความสนใจ]
[เนื้อหา — แบ่งย่อหน้าสั้นๆ]
[Emoji bullet ถ้ามี list]
[CTA — ถามคำถาม / กดไลค์ / แชร์ / คลิกลิงก์]
[#hashtag 3-5 ตัว]
```

## LINE OA Broadcast

### ข้อควรรู้
- จำกัดความยาว — คนไทยอ่าน LINE แบบเร็ว
- มีปุ่ม Rich Message ได้
- ส่งรูปก่อนข้อความ = engagement สูงกว่า
- ไม่ควรส่งเกิน 2-3 ครั้ง/สัปดาห์

### Template
```
[รูป/Sticker ดึงดูด]
[ข้อความสั้น 2-3 บรรทัด]
[CTA — ลิงก์ / ปุ่ม / คูปอง]
```
