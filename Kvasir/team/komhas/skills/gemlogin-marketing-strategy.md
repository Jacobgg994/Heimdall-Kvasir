# GemLogin Marketing Strategy

## 1. Product Positioning Map

### ตำแหน่งทางการแข่งขัน
```
                    ราคาสูง
                      |
    Multilogin (€99)  |
                      |
    GoLogin ($24) ———●—  GemLogin  ← "Best Value + Automation + Marketplace"
                      |
    AdsPower ($9) ——  |
    Dolphin ($10)     |
                      |
                    ราคาต่ำ
```

**การตีความทางการตลาด:** GemLogin อยู่ในตำแหน่ง "Mid-Range with Highest Feature Density" — ราคากลาง ($8-20/เดือน สำหรับ Local) แต่ฟีเจอร์ครบกว่าคู่แข่งในระดับราคาเดียวกัน เพราะมี Marketplace + No-Code Automation + Cloud Sync ในตัวเดียว

### Unique Selling Propositions (USP)
1. **Anti-Detect + Automation ในที่เดียว** — คู่แข่งส่วนใหญ่มีแค่ Fingerprint ไม่มี Automation
2. **GemStore Marketplace** — ผู้ใช้ซื้อสคริปต์สำเร็จรูป 1 คลิก ไม่ต้องเขียนโค้ด
3. **70% Revenue Share** — ดึงดูด Developer ให้มาสร้าง Ecosystem (คู่แข่งไม่มี)
4. **ภาษาไทย** — Manual และ Support ภาษาไทยครบถ้วน (เจาะตลาดไทยได้ดีกว่าคู่แข่งต่างประเทศ)
5. **Local & Cloud** — มีทั้ง 2 แบบ ให้เลือกตามความเหมาะสม
6. **BoxPhone + GemPhoneFarm** — โซลูชั่นครบวงจร ตั้งแต่ Browser ถึงมือถือ

## 4. กลยุทธ์ Affiliate Marketing

### 4.1 โครงสร้าง Affiliate (จาก Manual)
| รายการ | รายละเอียด |
|--------|------------|
| ค่าคอมมิชชั่น | 20% ของยอดสั่งซื้อ |
| เงื่อนไขรับเงิน | มียอดสั่งซื้อ ≥ 5 ราย และสะสม ≥ $100 |
| ระยะเวลา Commission | ภายใน 30 วันนับจากวันที่สมัคร |
| รอบจ่ายเงิน | วันที่ 22 ของเดือน (ตัดรอบ 15) |
| ค่าธรรมเนียมถอน | ไม่มี |

(ที่มา: https://manual.gemlogin.io/information/gemlogin/user-affiliate.md)

### 4.2 กลยุทธ์สำหรับ Affiliate
1. **สอนก่อนขาย:** สร้าง Content สอนใช้ GemLogin ฟรี → ปิดท้ายด้วย Link Affiliate
2. **เปรียบเทียบแพ็กเกจ:** "แพ็กเกจไหนคุ้มที่สุด?" พร้อม Link Affiliate
3. **Tutorial Specific:** "วิธีใช้ GemLogin สำหรับ [Platform]" เช่น TikTok, Facebook, Shopee
4. **Review เจาะลึก:** "รีวิว GemLogin 2026 ใช้จริง 30 วัน"
5. **กลุ่มเป้าหมาย Affiliate:** เจาะกลุ่มนักการตลาดออนไลน์, Freelancer, เจ้าของธุรกิจ SME

**การตีความทางการตลาด:** 20% Commission สูงกว่า標準 (ปกติ 5-15%) และไม่มีค่าธรรมเนียมถอน ทำให้ดึงดูด Affiliate ได้ดี เงื่อนไข 5 ราย/$100 เป็น Gate ที่พอเหมาะ ไม่สูงเกินไป

## 5. กลยุทธ์ GemStore Marketplace

### 5.1 โครงสร้าง Marketplace
- **Revenue Share:** Developer ได้ 70% (สูงผิดปกติในวงการ)
- **หมวดหมู่ Script:** Social Media, E-commerce, Scraping, Affiliate Marketing
- **ประเภท Script:** ฟรี / เสียเงิน
- **เปิดร้านค้าได้ที่:** store.gemlogin.io
- **ระบบป้องกัน:** Encrypt Script (ป้องกันแก้ไข) / Can Edit (เปิดให้แก้ไขได้)

(ที่มา: https://manual.gemlogin.io/information/gemlogin/marketplace-and-gem-store.md)

### 5.2 กลยุทธ์ Ecosystem
1. **Recruit Developers:** หานักพัฒนามาทำ Script ขาย — ค่าตอบแทน 70% ดึงดูด
2. **Quality Control:** ตรวจสอบ Script ก่อน Publish (ระบบ Pending Review)
3. **Top Sellers Contest:** จัดประกวด Script ยอดนิยมประจำเดือน
4. **Free Scripts for Marketing:** ทำ Script ฟรีคุณภาพดี ดึงดูดผู้ใช้ใหม่
5. **Script Bundles:** ขายเป็นชุด (Bundle) ราคาพิเศษ

## 6. กลยุทธ์ Pricing & Packaging

### 6.1 จุดขาย Local Storage
- **Entry Level ($0-8/mo):** 5-50 โปรไฟล์ ใช้งานส่วนตัว
- **Mid Level ($13-15/mo):** 200-1000 โปรไฟล์
- **Unlimited ($20/mo):** ไม่จำกัดโปรไฟล์ คุ้มมากสำหรับมืออาชีพ
- **Lifetime ($140):** จ่ายครั้งเดียวใช้ตลอดชีพ

### 6.2 จุดขาย Cloud Storage
- **Starter ($19/mo):** 10 โปรไฟล์ + 5 Threads
- **Professional ($99/mo):** 100 โปรไฟล์ + 25 Threads
- **Enterprise (Custom):** ไม่จำกัดโปรไฟล์ + 50 Threads + On-premise

### 6.3 กลยุทธ์ Upsell
1. **Free → Local:** 5 โปรไฟล์เต็ม → แนะนำ GEM50 ($8/เดือน)
2. **Local Small → Local Unlimited:** 200/500 โปรไฟล์ใกล้เต็ม → แนะนำ Unlimited ($20)
3. **Local → Cloud:** ขาย Cloud Sync สำหรับทีม หลายเครื่อง
4. **Lifetime:** เน้นขายให้คนที่ใช้ระยะยาว (คุ้มค่าใน 7 เดือน)
5. **GemLogin → GemPhoneFarm:** ทำ Cross-Sell (ใช้อันนึงก็ต้องใช้อีกอัน)

## 7. Campaign Ideas

### 7.1 "Anti-Detect Month" (ทุกเดือน)
- โปรโมทความสำคัญของ Anti-Detect Browser
- Free Trial 7 วัน สำหรับ Local Unlimited

### 7.2 "Automation Challenge"
- ผู้ใช้ส่ง Workflow ที่สร้าง
- คัดเลือกและแชร์ใน Social Media
- รางวัล: ส่วนลด, Script ฟรี, เครดิต

### 7.3 "GemStore Launch" (สำหรับ Developer)
- โปรโมท Revenue Share 70%
- Workshop การสร้าง Script
- Developer Grant สำหรับ Script คุณภาพ

- ส่วนลด 20% สำหรับการอัปเกรดจาก Local → Cloud
- ทดลองใช้ Cloud ฟรี 14 วัน
- Webinar "การจัดการทีมด้วย Cloud Storage"
