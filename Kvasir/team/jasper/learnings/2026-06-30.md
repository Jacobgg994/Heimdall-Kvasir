## 🔥 สิ่งที่ได้เรียนรู้

### TheGP/untidetect-tools
- **คืออะไร:** คลังทรัพยากรต่อต้านการตรวจจับ (anti-detection) ที่รวบรวมเบราว์เซอร์ปลอมลายนิ้วมือ, ไลบรารีอัตโนมัติ, บริการ captcha/SMS, และ proxy ไว้ในที่เดียว — ไม่ใช่โค้ดแต่เป็นคู่มือรายการแบบมีคำวิจารณ์
- **สิ่งที่ได้:**
  - รู้จักเบราว์เซอร์ anti-detect ดังๆ หลายตัว — GoLogin, Multilogin, AdsPower, Dolphin-anty, Camoufox — พร้อมราคาและข้อดีข้อเสีย
  - มีรายการเว็บทดสอบ fingerprint เยอะมาก — CreepJS, Pixelscan, BrowserLeaks, Rebrowser-bot-detector — ใช้ตรวจสอบว่า automation tools ตรวจจับได้มากแค่ไหน
  - Anti-detection ต้องวางหลายชั้นพร้อมกัน — เบราว์เซอร์ + proxy + captcha solver + SMS verification + humanizing tools (ขยับเมาส์แบบคนจริง) ตัวเดียวไม่พอ
  - มีคำเตือนว่าเครื่องมือหลายตัวล้าสมัยเร็วมาก — puppeteer-extra-stealth โดนเรียกว่า "trash, as easily detected by CreepJS"
  - Worker scope เป็นช่องโหว่ของการตรวจจับที่ extension ธรรมดาเข้าไปแก้ไข object properties ใน worker ไม่ได้
- **นำไปใช้ยังไง:** เอารายการ detection test sites มาใช้ตรวจสอบ farm phone setups และ gemlogin profiles ว่าปลอดภัยแค่ไหนก่อนปล่อยลง production

### enaqx/awesome-react
- **คืออะไร:** รายการเครื่องมือและไลบรารีของ React ecosystem ที่คัดสรรมาแล้ว — ตั้งแต่ framework, state management, component library, ไปจนถึง testing, animation, charts, และ renderers พิเศษ
- **สิ่งที่ได้:**
  - สายพันธุ์ state management ปัจจุบันมีเยอะมาก — จาก Redux/MobX สู่ Zustand, Jotai, TanStack Query — รู้ว่าตัวไหนกำลังมาแรงและเหมาะกับงานประเภทไหน
  - Component libraries ที่เป็นมาตรฐานตอนนี้คือ shadcn-ui (Radix UI + Tailwind), ตามด้วย Ant Design และ Mantine
  - มี React renderers ที่น่าสนใจ — react-three-fiber (3D), Ink (CLI), Remotion (video), react-pdf (PDF), react-figma (Figma plugin)
  - Testing stack มาตรฐานคือ Jest + React Testing Library + Playwright — ครบทั้ง unit, integration, E2E
  - รู้เรื่อง React Native ecosystem ด้วย — navigation, vector icons, device info, maps
- **นำไปใช้ยังไง:** ใช้เป็น reference เวลาต้องออกแบบ dashboard หรือ UI สำหรับ internal tools (Covertia, PKrist) — เลือก component library และ state management approach ได้ตรงกับความต้องการ

### shixinzhang/tiktok-viral-hooks
- **คืออะไร:** คลังถอดรหัส TikTok ที่ติดเทรนด์ แยกแยะว่าแต่ละคลิปใช้ hook แบบไหน, retention จุดไหน, และสรุปเป็น 2-sentence formula ที่ copy ได้
- **สิ่งที่ได้:**
  - รู้จัก hook patterns มากกว่า 83 แบบ — ตั้งแต่ Curiosity Gap, Rhetorical Question, Challenge, Contrast, Confession, ไปจนถึง Subtle Flex และ Over-the-Top Absurdity
  - แต่ละคลิปถูกวิเคราะห์เป็นระบบ — เปิดกี่วินาทีแรก promise อะไร, 7-14 วินาทีทำอะไรให้คนดูไม่หนี, อัลกอริทึมชอบตรงไหน
  - มีการจัดหมวดตาม niche — Entertainment (65), Beauty (14), Food (12), Finance (10), Tech (3) — ช่วยให้เลือกเรียนเฉพาะแนวที่เกี่ยวข้อง
  - มี mind map เป็น Mermaid diagram ของ script structure — มองเห็นภาพรวมว่าคลิปยาว 1-2 นาทีมีโครงสร้างแบบไหน
  - การจัดเก็บ formula เป็น 2-sentence template ทำให้เอาไปปรับใช้จริงได้ทันที ไม่ใช่แค่แนวคิดลอยๆ
- **นำไปใช้ยังไง:** ใช้เป็นคู่มือออกแบบ content strategy สำหรับงาน Thai market growth — สอน SALMON และทีม content ว่า hook แบบไหนเหมาะกับแต่ละกลุ่มเป้าหมาย, แต่ละ platform (TikTok/FB/LINE)
