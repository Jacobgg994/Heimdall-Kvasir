## 🛠️ วิธีเปิด Instagram Shop (Step-by-Step)

### คุณสมบัติ

```
✅ Business หรือ Creator Account
✅ ขาย Physical Products (ของจับต้องได้)
✅ มี Facebook Page ที่เชื่อมอยู่
✅ อยู่ในประเทศที่รองรับ (ไทย = ✅)
✅ Verified Domain ใน Business Manager
```

### Step 1: เปลี่ยนเป็น Professional Account

```
Settings → Account → Switch to Professional Account
→ เลือก Business หรือ Creator
```

### Step 2: เชื่อม Facebook Page

```
Settings → Business → Connect Facebook Page
→ เลือก Page ที่ใช่
```

### Step 3: สร้าง Product Catalog

```
ไปที่ commerce.facebook.com → Commerce Manager
→ Create Catalog → เลือก E-commerce
→ เชื่อม Shopify / WooCommerce / Upload CSV
```

| วิธีอัปโหลด | เหมาะกับ |
|------------|---------|
| **Shopify Sync** | มีเว็บ Shopify → อัตโนมัติ |
| **WooCommerce Sync** | มีเว็บ Woo → Plugin |
| **CSV Manual** | <100 สินค้า |
| **Pixel + Catalog Feed** | Programmatic |

### Step 4: Verify Domain

```
Business Settings → Brand Safety → Domains → Add → Verify
```

### Step 5: Submit for Review

```
⏰ รีวิวใช้เวลา 24-48 ชม. (นานสุด 4 สัปดาห์)
📌 Meta ตรวจ: สินค้าไม่ละเมิดนโยบาย, Domain Verified, Catalog ครบ
```

### Step 6: เปิด Shopping

```
Settings → Business → Shopping → Enable
→ เลือก Catalog → Done
```

## 🏷️ Product Tagging — กฎที่ต้องรู้

| Format | Tag สูงสุด | Best Practice |
|--------|----------|---------------|
| **Feed Post (รูปเดี่ยว)** | 5 | Tag 2-3 ตัวเด่น — ไม่ต้อง Tag ทุกอย่าง |
| **Carousel** | 20 (ทุกสไลด์รวมกัน) | เล่าเรื่อง — "3 วิธีใส่ตัวนี้" |
| **Reels** | 30 | **Tag ก่อน Publish** — หลังโพสต์แก้ไข Tag ไม่ได้ |
| **Stories** | 5 (ผ่าน Sticker) | วางข้างสินค้า ไม่ทับ |
| **Live** | ไม่มี Tag | ใช้ Pin Comment + Link Sticker |

### Product Title — SEO บน IG

```
❌: "เสื้อสีฟ้า"
✅: "เสื้อเชิ้ตแขนยาวผู้ชาย ผ้า Oxford สีกรม — Cotton 100%"

สูตร: Brand + Gender + Product Type + Key Attribute + Material
```

## 📸 7 เทคนิคทำให้ Shop ขายดี

### 1. รูปสินค้า — Instagram Quality

```
📸 รูปหลัก: พื้นขาว / สะอาด
📸 รูป 2: คนใส่/ใช้จริง (Model)
📸 รูป 3: Close-up — รายละเอียด
📸 รูป 4: Lifestyle — ในชีวิตจริง
📸 รูป 5: Size Comparison — เทียบกับของที่รู้จัก
```

### 2. Caption = SEO + CTA

```
❌: "ใหม่! กดลิงก์ใน Bio"
✅: "เสื้อเชิ้ตแขนยาว Oxford เนื้อผ้าเบา ใส่ได้ทั้งทำงานและวันสบายๆ 
      มี 5 สี ราคา 890 บาท 
      📍 กดที่รูปเพื่อดูShop หรือคอมเมนต์ 'สนใจ' ได้เลยค่ะ"
```

### 3. Reels = Discovery Engine

```
Reels ได้ Reach มากกว่า Feed Post 36%
→ Tag สินค้าใน Reels ก่อน Publish
→ Hook: "3 วิธีใส่เสื้อตัวนี้ให้ดูแพง"
```

### 4. Stories = ปิดการขาย

```
Story 1: รีวิวจากลูกค้าจริง
Story 2: Poll — "ชอบสี A หรือ B?"
Story 3: Countdown — "Flash Sale พรุ่งนี้!"
Story 4: Product Sticker → "กดซื้อเลย"
```

### 5. UGC (User-Generated Content)

```
ขอลูกค้าถ่ายรูปใช้สินค้า → Repost
= Social Proof + Content ฟรี + ลูกค้ารู้สึกพิเศษ
```

### 6. Collection — จัดหมวดหมู่

```
📂 "ของใหม่"
📂 "ขายดี"
📂 "ลดราคา"
📂 "เซ็ตประหยัด"
📂 "ของขวัญ"
```

### 7. Instagram Live — ขายสด

```
Live → โชว์ของจริง → ตอบคำถาม → 
Pin Comment: "ทัก DM คำว่า SHOP" → ปิดการขายใน DM
```

## 🔗 เชื่อม Instagram Shop กับ LINE OA

```
IG Post/Reels → Tag Product → 
ลูกค้ากด Tag → เห็นรายละเอียด → 
กดลิงก์ไปเว็บ → หรือทัก DM →
DM Auto-Reply: "สนใจตัวนี้แจ้ง LINE OA ได้เลย 👉 [ลิงก์]" →
LINE OA → ส่ง QR PromptPay → จบ
```

## ⚠️ 7 ข้อผิดพลาด

| ❌ ผิด | ✅ ถูก |
|--------|-------|
| Tag สินค้าใน Reels หลัง Publish (ทำไม่ได้) | Tag ก่อน Publish |
| รูปสินค้าไม่ชัด | 5 รูป — ชัด, Close-up, ใส่จริง |
| ไม่มีเว็บ — หวังขายใน IG อย่างเดียว | มีเว็บตัวเอง — IG = Discovery |
| ไม่ Verified Domain | Verified Domain = Throughput |
| Catalog ไม่ Update — ของหมด | Update Stock ทุกสัปดาห์ |
| Caption SEO แย่ | Keywords + Product Type + Attribute |
| ไม่มี UTM — วัดไม่ได้ | ทุกลิงก์ติด UTM |
