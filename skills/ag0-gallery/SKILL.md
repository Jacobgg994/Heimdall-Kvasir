## 18 สไตล์การสร้างรูป/วิดีโอ

### 🎬 ภาพยนตร์ (Cinematic)
สร้างภาพแบบฟิล์มหนัง — moody lighting, ดราม่า, โทนสีภาพยนตร์

**เทคนิค:** ใส่คำว่า "cinematic", "film-like", "movie poster", "dramatic lighting"
```
ตัวอย่าง: "โปสเตอร์แฟชั่นลึกลับ สไตล์ภาพยนตร์" [ChatGPT]
ตัวอย่าง: "สาวเกาหลีริมหน้าต่างสนามบิน: ภาพยนตร์สารคดี" [ChatGPT]
```

### 📸 ภาพถ่าย (Photography)
แนวกล้องถ่ายรูป — CCD, DSLR, ฟิล์ม, เซลฟี่, ภาพถ่ายบุคคล

**เทคนิค:** ใส่คำว่า "photography", "CCD", "selfie", "camera shot", "DSLR quality"
```
ตัวอย่าง: "ภาพเดทสไตล์ CCD โทนเย็นสดใสในเมือง" [ChatGPT]
ตัวอย่าง: "เซลฟี่สาวชุดดำ สไตล์เรียบง่ายในร่ม" [Nano Banana 2]
```

### ✨ สมจริงสุดๆ (Hyper-realistic)
ภาพที่แทบแยกไม่ออกจากรูปถ่ายจริง — ผิวละเอียด, แสงเงาแม่นยำ

**เทคนิค:** ใส่ "hyper-realistic", "8K", "photorealistic", "intricate detail"
```
ตัวอย่าง: "ภาพพอร์ตเทรตหญิงสาวเอเชีย แนว CCD สไตล์ Hyper-realistic" [ChatGPT]
```

### 👗 แฟชั่น (Fashion)
แฟชั่นทุกรูปแบบ — streetwear, runway, minimal fashion, Y2K

**เทคนิค:** ระบุสไตล์แฟชั่นชัดๆ + ใส่บริบท (location, lighting, mood)
```
ตัวอย่าง: "ภาพบุคคลแฟชั่นบนพื้นหญ้า ถ่ายมุมสูงสไตล์ Y2K" [ChatGPT]
ตัวอย่าง: "สตรีทแฟชั่นสุดหรู ย่านช้อปปิ้งเมืองใหญ่" [ChatGPT]
```

### 🌿 ธรรมชาติ (Nature)
ทิวทัศน์, กลางแจ้ง, beach, ป่า, ดอกไม้, golden hour

**เทคนิค:** ใส่ฤดูกาล + เวลา + สภาพอากาศ + แสง
```
ตัวอย่าง: "สาวชายหาดสุดโรแมนติกในวันแดดใส" [ChatGPT]
ตัวอย่าง: "หญิงสาวในป่าสน: ภาพบุคคลสวยงามท่ามกลางแสงสนธยา" [ChatGPT]
```

### 👤 ภาพบุคคล (Portrait)
พอร์ตเทรต — ใบหน้า, close-up, headshot, ทั้งในสตูดิโอและ outdoor

**เทคนิค:** ระบุอารมณ์ + แสง + พื้นหลัง + สไตล์เสื้อผ้า
```
ตัวอย่าง: "สาวเอเชียในทุ่งหญ้า สวยใส สไตล์ฤดูร้อน" [ChatGPT]
ตัวอย่าง: "สาวจีนใหม่ในฤดูร้อน กำลังเล่นว่าว แบบ POV สมจริง" [ChatGPT]
```

### 💎 หรูหรา/พรีเมียม (Luxury)
สินค้าหรู, ดีไซน์หรู, ทอง, หินอ่อน, crystal, แบรนด์ premium

**เทคนิค:** ใส่วัสดุหรู + จัดแสง dramatic + gold/marble/velvet
```
ตัวอย่าง: "โปสเตอร์โฆษณาสินค้าหรูหรา สไตล์ภาพยนตร์" [ChatGPT]
ตัวอย่าง: "โลโก้เลนส์พรีเมียม ขาวดำ ไฮคอนทราสต์ มินิมอล" [ChatGPT]
```

ตัวอย่าง: "อนิเมะญี่ปุ่นยุค 80-90: ภาพประกอบสไตล์เซลลูลอยด์" [ChatGPT]

## หลักการเขียนพรอมต์ให้ได้ผล

### 1. ระบุ Subject ชัดเจน
```
❌ "รูปผู้หญิงสวย"
✅ "สาวเอเชียตะวันออก อายุ 25 ปี ผมยาวสีดำ ใส่ชุดเดรสสีแดง 
    ยืนอยู่ริมชายหาดตอนพระอาทิตย์ตก แสง golden hour"
```

### 2. ใส่สไตล์ภาพ (Style Reference)
```
❌ "รูปอาหาร"
✅ "อาหารญี่ปุ่น ถ่าย style food photography แสงธรรมชาติ 
   จัดจานแบบ minimal มุม 45 องศา depth of field ตื้น"
```

### 3. ระบุ Mood + แสง
```
❌ "ผู้หญิงในห้อง"
✅ "ผู้หญิงในห้องนอน แสงเช้าอ่อนๆ ลอดผ่านม่านผ้าบาง 
   mood อบอุ่น สบายๆ เช้า lazy sunday morning"
```

### 4. ใส่มุมกล้อง + เลนส์
```
- "birds eye view", "top down"
- "eye level portrait"
- "wide angle", "fisheye"
- "85mm portrait lens", "bokeh background"
- "dutch angle", "low angle shot"
```

### 5. อ้างอิงสไตล์ช่างภาพ/ผู้กำกับ
```
- "Wes Anderson style" — สมมาตร, สีพาสเทล
- "Makoto Shinkai style" — ฟ้าสวย, แสงสวย, โรแมนติก
- "Studio Ghibli style" — อบอุ่น, ธรรมชาติ, มหัศจรรย์
- "Wong Kar-wai style" — ฟิล์ม, มู้ดดี้, วินเทจ
- "Annie Leibovitz style" — dramatic, cinematic portrait
```

### 6. ระดับรายละเอียด (Quality Tags)
```
- "8K", "ultra high resolution", "highly detailed"
- "award winning photography"
- "masterpiece", "best quality"
- "intricate textures", "sharp focus"
```

## การ Mix & Match สไตล์

ความสนุกของการสร้างรูป AI คือการผสมสไตล์ที่ต่างกัน:

```
✅ "ภาพบุคคล + อนิเมะ" = ภาพคนจริงแต่เป็นสไตล์ Ghibli
✅ "แฟชั่น + วินเทจ" = แฟชั่นสมัยใหม่ถ่ายแบบกล้องฟิล์มยุค 90
✅ "อาหาร + มินิมอล + หรูหรา" = fine dining minimalist
✅ "สถาปัตยกรรม + เหนือจริง" = ตึกลอยได้ เหนือจริง
✅ "ตัวละคร + สมจริงสุดๆ" = ตัวละครเกมที่เหมือนคนจริง
```

## Tips จากคลัง 12,989 พรอมต์

1. **ภาษาไทยใช้ได้ดี** — 80% ของพรอมต์ในคลังใช้ภาษาไทยเป็นหลัก
2. **ใส่ context เยอะๆ** — prompt ที่ยาวและมีรายละเอียดมากได้ผลดีกว่า prompt สั้น
3. **ระบุอารมณ์** — "สดใส", "ลึกลับ", "โรแมนติก", "อบอุ่น" — AI เข้าใจ mood words
4. **บอกสิ่งที่ไม่ต้องการด้วย** — "no watermark", "no text", "no blur"
5. **ทดลองโมเดล** — prompt เดียวกันกับคนละโมเดล ได้ผลลัพธ์ต่างกันมาก
6. **บันทึก prompt ที่ใช้ได้ผล** — เป็น object ไว้ใช้อ้างอิงภายหลัง
