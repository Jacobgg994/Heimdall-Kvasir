# 🚫 Content Rules & Decisions

## Removed / Banned
1. **QCCAP** — removed from all content plans (Product Lead position canceled)
2. **Discount broadcasts** — ห้าม broadcast ลดราคา (discount, coupon, โปรโมชั่นราคา)
3. **Greeting openings** — ห้ามเปิดด้วย "สวัสดี", "ยินดีต้อนรับ" — เปิดด้วย value ทันที
4. **Thank you fluff** — ห้ามจบเดือนด้วย "ขอบคุณ" หรือ "teaser" ลอยๆ
5. **Urgency tactics** — ห้าม "หมดเขต", "เหลือ X วัน", "รีบก่อน" — soft sell only

## Content Philosophy (decided 2026-06-30)
- **Balanced weekly mix:** Every week has 🔧 เทคนิค + 📚 ความรู้ + 🚀 ขาย
- **Ratio target:** 40% เทคนิค / 30% ความรู้ / 20% ขาย / 10% บันเทิง
- **Sell softly:** "มีของดี อยากให้ลอง" — ไม่ aggressive, ไม่ discount, ไม่ urgency
- **Thai-first:** Content หลักเป็นภาษาไทย (LinkedIn เป็น ENG)
- **Value over promotion:** ทุก content ต้องให้คุณค่าก่อน — ขายทีหลัง

## LINE OA Rules
- Max 3 broadcasts/week
- Time: 12:00 or 20:00
- Image before text always
- Rich Message button every time
- 2-3 lines max
- No formal/ราชการ language
